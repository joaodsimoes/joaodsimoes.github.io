Motion detector Server

Intro

After arriving in Slovenia I was ordered to quarantine for 10 days due to Covid-19 restrictions.
I was told health inspectors could come knocking on my door any time of the day to check on me.
Binging Netflix shows soon got old and a certain unrest grew on me, I got worried I wouldn't be able to hear someone knocking on the door, so I decided to mount an impromptu surveillance system thanks to an old Android phone my girlfriend had laying around and with the help of AlfredCamera Android app.

Demo

For the demo I setup two cameras, one pointing at the stairway leading to the front door and another showing the front door entrance and living room.

How it works

The userscript pulls a base 64 encoded videoframe every 400ms and sends it to the server.
Upon recieving a request, the server checks if the cameraID is already registered in the database, if it's not, it processes the frame and sets it as background.
If a background for the cameraID is already present in the database, the server pulls it and compares it to the current frame through Absolute Difference, if an object past a certain area threshold is detected, alarms are sounded, a picture with the intruder is saved to the filesystem and the server returns a 200 OK {Intruder: true} response.
